## 구현 시 어려웠던 점
- SNS 연결하는 기능을 구현할때 링크를 객채로 생성하고 운영체제 쉘에 도움을 받아야 한다는 점을 몰랐고 정보를 보내기 위해 객채를 쓴다는 점을 이해하는데 시간이 걸렸다.
- 사진 변경하는 기능을 구현할때 사진을 리소스파일에 등록하고 가져오는 구조를 한번에 이해하지 못했었다.
- 마우스 위치에 따라 사진 테두리를 변화시키는 기능을 구현할때 문법적으로 지식이 부족해 코드 작성할때 버그가 많았다.
